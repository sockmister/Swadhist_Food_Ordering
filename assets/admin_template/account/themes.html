Activate

To use the theme activation in php, you need to use the $_COOKIE['theme'] in the style url. e.g. css/ui/<?=$_COOKIE['theme']?>/ui.css

To create a new theme, just copy the existing theme under css/ui folder and give it a new name, and modify the css as you wish.
